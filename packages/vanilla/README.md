# @dmc--98/dfe-vanilla

Vanilla JavaScript bindings for Dynamic Form Engine with EventTarget-based reactivity

## Install

```bash
npm install @dmc--98/dfe-vanilla
```

## Overview

This package is part of the Dynamic Form Engine monorepo and is published independently so you can adopt only the pieces you need.
