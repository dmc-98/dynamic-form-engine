# @dmc--98/dfe-dashboard

Admin dashboard for Dynamic Form Engine — manage forms, submissions, analytics

## Install

```bash
npm install @dmc--98/dfe-dashboard react react-dom
```

Peer dependencies:

- `react`
- `react-dom`

## Overview

This package is part of the Dynamic Form Engine monorepo and is published independently so you can adopt only the pieces you need.
