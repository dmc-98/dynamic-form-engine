# @dmc--98/dfe-fastify

Fastify plugin for Dynamic Form Engine

## Install

```bash
npm install @dmc--98/dfe-fastify fastify
```

Peer dependencies:

- `fastify`

## Overview

This package is part of the Dynamic Form Engine monorepo and is published independently so you can adopt only the pieces you need.
