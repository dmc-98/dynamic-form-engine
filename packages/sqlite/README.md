# @dmc--98/dfe-sqlite

SQLite adapter for Dynamic Form Engine using better-sqlite3

## Install

```bash
npm install @dmc--98/dfe-sqlite better-sqlite3
```

Peer dependencies:

- `better-sqlite3`

## Overview

This package is part of the Dynamic Form Engine monorepo and is published independently so you can adopt only the pieces you need.
