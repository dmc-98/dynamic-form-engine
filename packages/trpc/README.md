# @dmc--98/dfe-trpc

tRPC integration for Dynamic Form Engine

## Install

```bash
npm install @dmc--98/dfe-trpc @trpc/server
```

Peer dependencies:

- `@trpc/server`

## Overview

This package is part of the Dynamic Form Engine monorepo and is published independently so you can adopt only the pieces you need.
