# @dmc--98/dfe-ui-mantine

Mantine UI components for Dynamic Form Engine

## Install

```bash
npm install @dmc--98/dfe-ui-mantine @mantine/core @mantine/hooks react react-dom
```

Peer dependencies:

- `@mantine/core`
- `@mantine/hooks`
- `react`
- `react-dom`

## Overview

This package is part of the Dynamic Form Engine monorepo and is published independently so you can adopt only the pieces you need.
