# @dmc--98/dfe-ui-shadcn

shadcn UI components for Dynamic Form Engine

## Install

```bash
npm install @dmc--98/dfe-ui-shadcn @radix-ui/react-dialog @radix-ui/react-label @radix-ui/react-popover @radix-ui/react-select @radix-ui/react-separator @radix-ui/react-tabs class-variance-authority clsx react react-dom tailwind-merge
```

Peer dependencies:

- `@radix-ui/react-dialog`
- `@radix-ui/react-label`
- `@radix-ui/react-popover`
- `@radix-ui/react-select`
- `@radix-ui/react-separator`
- `@radix-ui/react-tabs`
- `class-variance-authority`
- `clsx`
- `react`
- `react-dom`
- `tailwind-merge`
