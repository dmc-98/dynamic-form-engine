# Acknowledgements

Dynamic Form Engine exists because of a strong open-source ecosystem. This page is a thank-you to the tools and projects that make the repo possible.

## Core Building Blocks

- TypeScript for the language and type system the whole project is built around.
- React for the primary stable UI integration and Playground surface.
- Zod for schema generation and validation ergonomics.
- Prisma and Drizzle for the adapter-backed persistence story.
- Express and Hono for backend delivery paths across traditional and serverless runtimes.

## Developer Experience And Tooling

- VitePress for the docs site.
- Vitest for fast package-level testing.
- Playwright for real browser verification.
- Storybook for UI package development and review.
- Turborepo for monorepo task orchestration and caching.
- Changesets for release versioning and publish workflows.

## Ecosystem Inspiration

DFE also benefits from ideas proven by other form tools and ecosystems:

- React Hook Form
- Formik
- RJSF
- SurveyJS
- Form.io
- builder-style internal tools and workflow systems across the TypeScript ecosystem

This page is not an endorsement claim from those projects. It is simply recognition that good open-source products learn from each other.

## Community Thanks

Thank you to:

- everyone opening issues with concrete reproduction steps
- people testing early packages and example apps
- contributors improving docs, coverage, and integrations
- users comparing DFE honestly against existing form tooling

## Maintainer Note

If a major upstream package or project materially shapes DFE, keep this page updated. Good acknowledgements make the project more trustworthy, not less.
